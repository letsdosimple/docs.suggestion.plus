## What is Suggestion Plus?

Suggestion Plus is an anonymous suggestion box platform that helps you collect honest feedback from your team, classroom, or community without compromising their privacy.

## Platform Highlights

- **100% Anonymous** — No IP logging, no email required, zero data retention
- **Easy Setup** — Create an inbox in under 60 seconds, no technical skills needed
- **Multi-Platform** — Available on web, Android, and iOS
- **Real-time Notifications** — Push notifications and dashboard alerts
- **QR Code Sharing** — Generate and print QR codes for physical locations
